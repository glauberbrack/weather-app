## :notebook: Running

### Requeriments

```bash
# Clone this
$ git clone https://github.com/glauberbrack/star-wars-app

# Go to your project folder
$ cd star-wars-app

# Install all depencies
$ yarn

# Run your project
$ yarn android
```

### Tests
```bash
# To check about the tests
$ yarn test
```
